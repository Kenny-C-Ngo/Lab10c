Lab 10c: Creating A Webpage

Installing The Framework

The framework that I chose to create the page was React. First I installed the appropriate version of nodejs for my computer from "www.nodejs.org". Next I created a folder in my drive to store my react projects. Then I opened powershell and changed my directory to the folder I created and executed the command "npx create-react-app lab10c" to install react and create my project. After that I used Visual Studio Code to start creating my website. Once I finished I entered my project's directory and ran the "npm start" command.
